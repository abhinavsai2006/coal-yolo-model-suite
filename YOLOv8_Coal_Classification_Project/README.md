# YOLOv8 Coal Classification Project

This folder contains all scripts, data, model weights, and reports for the YOLOv8-based coal classification system.

## Structure
- `scripts/` — All Python scripts for training, evaluation, metrics, dashboard, and PDF generation
- `data/` — The coal classification dataset (organized for YOLOv8 classification)
- `model/` — Trained YOLOv8 model weights
- `reports/` — All generated reports, PDFs, and visualizations

## Quick Start
1. Install requirements: `pip install -r requirements.txt`
2. See `scripts/` for all code
3. See `reports/` for results and documentation
